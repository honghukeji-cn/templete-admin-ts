# 环境建议
### node 版本 v20.15.0
### npm 版本 10.7.0
### yarn 版本 1.22.22
# 安装及运行
安装命令
### `yarn`
启动命令
### `yarn start`
打包命令
### `yarn build`

# 开发
## 路径说明
.env.development 设置开发时请求变量
.env.production 设置生产环境请求变量
## 色调规范
开发中请严格使用colorError、colorPrimary、colorSuccess、colorWarning 四个颜色变量，详见[reset.css](./src/reset.css)
